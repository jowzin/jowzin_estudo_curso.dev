# jowzin_estudo_curso.dev-1

Projeto de estudo com [Next.js](https://nextjs.org/) e React, criado no contexto do curso.dev.

## Tecnologias

- [Next.js](https://nextjs.org/) 13
- [React](https://react.dev/) 18

## Pré-requisitos

- [Node.js](https://nodejs.org/) (versão 16 ou superior recomendada)
- npm (vem com o Node.js)

## Como rodar

1. Clone o repositório e entre na pasta do projeto:

```bash
cd jowzin_estudo_curso.dev-1
```

2. Instale as dependências:

```bash
npm install
```

3. Inicie o servidor de desenvolvimento:

```bash
npm run dev
```

4. Abra [http://localhost:3000](http://localhost:3000) no navegador.

## Scripts disponíveis

| Comando      | Descrição                          |
| ------------ | ---------------------------------- |
| `npm run dev` | Inicia o app em modo desenvolvimento |
